# 跨链桥
链，Dapp 开发者，用户 都有 跨链的需求。
* 链：通过跨链获得其他链的生态。比如，有的链有专门的关注点：性能，安全，隐私等。
* Dapp 开发者：获得不同链上的用户。
* 用户：在一个链上，用不同链上的app，资产在不同链上转移。

跨链一般转移的都是资产(代币) 。也会有 NFT。也会有信息，事件等。

## 技术方案
保证跨链桥的安全性。跨链桥常采用外部中继人的模式。
